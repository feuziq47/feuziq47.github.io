ListToDict

sun  
파이썬 List to Dict 1. Dictionary Comprehesion A = ['가', '나', '다'] dictionary = {string: i for i, num in enumerate(A)} print(dictionary) # {'가':0 , '나':1, '다':2} 2. dict.fromkeys(key, value) A = ['가', '나', '다'] dictionary = dict.fromkeys(A, 0) print(dictionary) # {'가':0 , '나':0, '다':0} 3. zip 사용하여 묶기 A = ['가', '나', '다'] B = [0, 1, 2] dictionary = dict(zip(A, B)) # {'가':0 , '나':1, '다':2}

(둘 중 하나만 넣어도 성공…) 기술블로그 운영 IT업계에 종사하는 사람들이 다양한 이유로 블로그를 운영한다. 남들 하는거 다해봐야 하는 나도 기술블로그 운영에 도전한다.
